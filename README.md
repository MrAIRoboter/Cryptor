# Информация
Cryptor - это программа для шифрования или дешифрования всех файлов в указанной папке.

## Использование
```bash
cryptor [/e | /d] [пароль] ["путь"] [hdd | ssd]
```
- /e - Шифрование
- /d - Дешифрование
- hdd - жёсткий диск, ssd - твердотельный SSD накопитель (по-умолчанию hdd)

## Алгоритм шифрвоания
AES256-CBC
